# tomato-work-server

tomato-work-server 是 [tomato-work](https://github.com/xjh22222228/tomato-work) 的服务端。

## Build Setup

``` bash
# 安装依赖包
npm i

# 启动 Port: 7003
npm run dev

# 部署
npm start
```
